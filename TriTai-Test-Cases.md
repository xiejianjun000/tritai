# TriTai 三才 WFGY 测试用例体系 V0.1

## 测试设计原则

**正向测试**：应该被检测出的幻觉
**反向测试**：应该被检测但可能漏检的隐蔽幻觉
**逆向测试**：正常内容不应被误判
**边界测试**：临界值附近的表现
**组合测试**：多规则同时触发的场景
