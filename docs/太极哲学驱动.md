# 三才·太极哲学驱动

## 定位澄清

**三才是太极开源平台上的防虚幻技能，不是平台本身。**

```
太极开源（平台）
├── 四象安全架构：青龙/朱雀/玄武/白虎
├── 十二神智能体
├── 梦境系统：五阶段进化
└── 三才技能（本文档范围）← 技能层面
```

---

## ☯️ 三才的阴阳之道

### 三才名称由来

**三才 = 天时·地利·人和**

| 三才 | 含义 | 三才技能对应 |
|------|------|-------------|
| 天时 | 时机、趋势 | 时间穿越检测（R003） |
| 地利 | 事实、依据 | 知识图谱验证 |
| 人和 | 一致性、共识 | 自一致性检查 |

### 阴阳关系

**三才内部的阴阳**：

- **阳（WFGY）** = 符号层规则 = 确定性检测
- **阴（知识库）** = 动态知识 = 可更新的内容

```
WFGY规则（阳）←→ 知识库（阴）
     ↓              ↓
  确定性边界     可进化内容
     ↓              ↓
   锁定核心      订阅更新
```

**不是LLM的阴阳**，LLM是三才服务的对象，不是三才内部的阴。

## 三才的核心哲学

### 天时：时机检测

**时间穿越、历史幻觉**

```
R003 时间穿越：未来年份当作已发生
R006 虚假历史：错误的历史发布时间
```

### 地利：事实依据

**知识图谱、事实验证**

```
知识图谱：提供事实锚点
溯源索引：追溯知识来源
```

### 人和：一致性共识

**自一致性检查**

```
多次采样 → 投票 → 一致结果
```

## 三才的进化哲学

### 不是阴阳螺旋，是天时地利人和的循环

```
天时（检测）→ 发现幻觉
    ↓
地利（验证）→ 查证事实
    ↓
人和（进化）→ 规则优化
    ↓
天时（新检测）→ 循环
```

### 三层进化

| 层级 | 内容 | 进化方式 |
|------|------|----------|
| 天时层 | 核心规则 | 锁定，不进化 |
| 地利层 | 知识库 | 订阅更新 |
| 人和层 | 用户偏好 | 个人进化 |
