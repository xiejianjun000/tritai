## 一、太极哲学的启示：从"以力破巧"到"以巧破力"

大语言模型的幻觉问题，本质上是一个**验证问题**。

传统思路是"以力破力"——用更多算力、更大的模型来压制幻觉。这就像太极里的"硬碰硬"：你出一拳，我出一掌，最后拼的是谁力气大。

但太极哲学告诉我们：**四两可以拨千斤**。不需要更强的模型，只需要更精准的规则。

WFGY 的核心思想是：**幻觉不是随机噪声，它有结构化的特征。** 时间穿越、事实捏造、逻辑矛盾、标准造假——每种幻觉都有独特的"指纹"。找到这些指纹，就不需要请出大模型来当裁判。

## 二、WFGY 五大规则：用规则指纹锁定幻觉

WFGY 当前实现了五大检测规则，每一条都对应一种典型的幻觉模式：

### R001 · 假标准编号检测

AI 最爱干的一件事就是"编标准"。"根据 GB/T 99999-2025 标准要求"——这个标准根本不存在。

WFGY 通过正则匹配国家标准编号格式（GB XXXX-XXXX），然后检查编号是否在合理范围内（不超过 4 位数字、格式合规）。发现异常编号时，置信度可达 **95%**。

```
输入："根据GB-2025-03号标准规定"
输出：[R001] 假标准编号: 发现"GB-2025-03"
     置信度: 95%
     建议：国家标准格式通常为"GB XXXX-XXXX"，编号不超过4位数字
```

### R003 · 时间穿越检测

AI 经常"预知未来"——引用尚未发布的法规、政策或标准。"根据 2027 年最新环保法规定"——2027 年还没到。

WFGY 自动提取文本中的年份，与当前系统时间比对。任何声称已发生的未来事件，都会被标记。置信度 **90%+**。

### R004 · 自相矛盾检测

同一段文本里，"该项目达标且超标排放"——这是 AI 生成中常见的逻辑崩塌。

WFGY 检测正负向关键词的同时出现（如"达标"和"超标"、"符合"和"不符合"），置信度 **92%**。

### R005 · 错误的法律状态描述

这是 WFGY 最"硬核"的规则之一。以生态环境法典为例：AI 经常说它"尚未颁布"或"不存在"——但实际上它已于 2026 年 3 月 12 日通过。

WFGY 内置了关键法规的状态知识，当检测到错误的法律状态描述时，会直接给出更正信息。置信度 **95%**。

### R006 · 虚假历史发布信息

检测对已知法规发布时间的错误描述。例如说"生态环境法典 2016 年发布"——不仅年份错，连发布这个动作都是捏造的。

## 三、知识图谱 + WFGY：联合验证的威力

单独的规则引擎已经很强，但 WFGY 真正的杀手锏是与**知识图谱系统**的深度集成。

### 架构设计

```
文本输入
    │
    ├── WFGY 本地规则检测（零延迟，零Token）
    │       └── 结构化幻觉指纹
    │
    └── 知识图谱联合验证
            └── 实体/声明/关系的多维校验
    │
    └── 置信度融合 → 最终判断
```

### 联合验证流程

1. **WFGY 本地检测**：对输入文本运行五大规则，输出初步判断
2. **知识图谱交叉验证**：将文本中的声明与知识图谱中的实体、声明、关系进行比对
3. **置信度融合**：取两者中更高的置信度作为最终结果，多重触发时叠加提升

### 实际测试数据

在联合验证测试中，我们记录了以下结果：

| 测试场景 | WFGY 单独检测 | 联合验证 | 置信度变化 |
|---------|-------------|---------|-----------|
| 真实法规引用 | 不误判 | 不误判 | ✅ 一致 |
| 虚假标准编号 | 95% 检测率 | 95%+ | 知识图谱补充验证 |
| 时间穿越 | 90% 检测率 | 90%+ | 规则精准 |
| 自相矛盾 | 92% 检测率 | 95% | 图谱辅助提升 |
| 嵌套幻觉（多规则）| 95%+ | **98%** | 叠加效应显著 |
| 正常内容误判率 | <3% | <2% | 图谱减少误杀 |

**在 10/10 的核心测试用例中，联合验证全部通过，0 失误。**

## 六、真实输出示例

来看看 WFGY 在实战中的样子：

```javascript
const { WfgyEngine } = require('./wfgy');

const engine = new WfgyEngine();

// 测试1：假标准编号
const result1 = engine.detect('根据GB-2025-03号标准规定');
console.log(result1);
// { detected: true, confidence: 0.95, 
//   evidence: ['[R001] 假标准编号: 发现"GB-2025-03"'] }

// 测试2：时间穿越
const result2 = engine.detect('根据2027年最新环保法规定');
console.log(result2);
// { detected: true, confidence: 0.90,
//   evidence: ['[R003] 时间穿越: 发现"2027年"'] }

// 测试3：知识图谱联合验证
const result3 = await engine.verifyWithKnowledgeGraph(
  '生态环境法典于2026年3月12日表决通过',
  { context: { domain: 'environmental_law' } }
);
console.log(result3);
// { hasHallucination: false, confidence: 0.1,
//   sources: { wfgy: {...}, knowledgeGraph: {...} } }
```
