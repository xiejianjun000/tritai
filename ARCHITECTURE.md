# Tritai 三才 - 系统架构
> 零 Token AI 防幻觉引擎，基于太极哲学的 LLM 输出验证系统

## 系统概述

Tritai（三才）是一个基于太极哲学的 LLM 输出验证系统，通过天、地、人三层架构实现零 Token 的 AI 防幻觉验证。

### 设计理念
- **太极哲学**：运用阴阳对立统一的哲学思想
- **零 Token 消耗**：无需额外 API 调用即可验证输出
- **知识图谱**：基于结构化知识进行语义对齐
- **规则引擎**：WFGY 规则引擎提供可配置的验证逻辑

## 核心架构

```
┌──────────────────────────────────────┐
│              Tritai 系统              │
├──────────────────────────────────────┤
│  天 (天时)  │  地 (地利)  │  人 (人和)│
├──────────────────────────────────────┤
│           WFGY 规则引擎              │
├──────────────────────────────────────┤
│      OpenAI / Claude 适配器          │
└──────────────────────────────────────┘
```

## 模块设计

1. **天（天时）**：时间维度验证
2. **地（地利）**：空间维度验证  
3. **人（人和）**：语义维度验证

## 技术栈

- TypeScript 6.0+
- Node.js 18+
- ESLint 9+ (flat config)
